08 / TROUBLESHOOTING MEMORY ‚Äî SYMPTOMS & FIXES
üí• PC WON'T BOOT / NO POST
POSSIBLE CAUSES:
RAM not fully seated in slot
RAM in wrong slot (not A1+B1)
Incompatible DDR generation
Mixed RDIMM + UDIMM
FIXES:
Reseat RAM (remove and reinsert firmly)
Test one stick at a time
Check motherboard manual for slot order
üíô RANDOM BLUE SCREENS (BSOD)
POSSIBLE CAUSES:
Faulty RAM stick
Mixed RAM speeds (running at slowest)
Overclocking instability (XMP/DOCP)
FIXES:
Run MemTest86 (memory diagnostic tool)
Reset BIOS to defaults
Test sticks individually to isolate bad one
üê¢ SLOW + DISK ALWAYS 100%
POSSIBLE CAUSES:
RAM is full ‚Üí thrashing (paging constantly)
Too many programs running simultaneously
FIXES:
Add more physical RAM
Close background applications
Check Task Manager ‚Üí Memory usage
üîß
DECISION LOCK ‚Äî READ THE SYMPTOM, KNOW THE FIX
No POST at all = RAM not seated or wrong type ‚Üí reseat first, one stick at a time
Boots but random BSOD = bad or mismatched RAM ‚Üí MemTest86, then swap sticks
Boots, runs, but crawls + disk maxed = thrashing ‚Üí buy more RAM, full stop
Exam trap: "disk constantly active + slow" is a RAM question, not a storage question.
